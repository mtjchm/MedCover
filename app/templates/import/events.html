Jak importovat akce
  1. Otevřete Google Sheet Dozory YYYY a stáhněte ho jako Microsoft Excel (.xlsx) (Soubor → Stáhnout → Microsoft Excel).
  2. Spusťte konverzní skript (vyžaduje Python 3 + pip install openpyxl): 
    python scripts/import_events.py --input "Dozory 2026.xlsx" --output import_events.json
  3. Otevřete výsledný soubor import_events.json v textovém editoru, obsah celý zkopírujte a vložte do pole níže.
  4. Klikněte na Zkontrolovat a zobrazit náhled. Před potvrzením importu budete moci upravit všechny akce v přehledné tabulce.
